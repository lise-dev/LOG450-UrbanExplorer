# Urban Explorer 

Une application mobile pour explorer et partager des lieux insolites en ville.

## Technologies utilisées
- React Native (Expo)
- Firebase (Authentification & Base de données)
- OpenStreetMap (Cartographie)
- OpenWeatherMap (Météo)

## Installation

1. **Cloner le repo :**
```bash
   git clone https://github.com/TON-USER/TON-REPO.git
   cd LOG450-UrbanExplorer/UrbanExplorer
```

2. **Installer les dépendances**
```bash
    npm install
```

3. **Lancer l'application**
```bash
    npx expo start
```

4. Scanner le QR Code avec Expo Go sur iOS/Android pour tester sur mobile
